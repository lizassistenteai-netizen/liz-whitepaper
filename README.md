**Site Oficial:** [liziabr.qzz.io](https://liziabr.qzz.io/)

Este repositório contém o **Whitepaper Oficial da LIZ**, uma inteligência artificial de **7.2 bilhões de parâmetros**, 100% brasileira, autônoma e soberana, desenvolvida pela **Liz AI Studio**.

## 🚀 Sobre a LIZ

A LIZ não é apenas uma interface de chat; é um ecossistema neural de larga escala. Diferente de outras soluções, a LIZ é um modelo de linguagem proprietário (Zero-Dependency), treinado do zero com foco na soberania digital do Brasil.

### 🇧🇷 Pilares da LIZ
- **Soberania Digital**: Processamento e dados 100% em solo nacional.
- **Modelo Autônomo (7.2B)**: Treinado integralmente pela Liz AI Studio.
- **Scientific Specs**: Arquitetura Transformer Decoder-only (SwiGLU, RoPE, GQA).
- **Independência de Datasets**: 1.4 Trilhões de tokens otimizados para o contexto brasileiro.
- **Infraestrutura HPC**: Clusters NVIDIA H100 operando com energia renovável.
